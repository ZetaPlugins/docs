## Features

-   ✅ Custom links
-   ✅ Colored link Names
-   ✅ HEX Colors
-   ✅ Link commands (e.g. `/discord`, `/website`)
-   ✅ Highly customizable
-   ✅ Easy setup
-   ✅ Multiple languages
